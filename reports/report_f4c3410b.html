📊 Approach & Query

Query Code

SELECT dsg.region, dp.category, SUM(fsg.total_amount) as total_sales_revenue
FROM fact_sales_gold fsg
JOIN dim_products_gold dp ON fsg.product_id = dp.product_id
JOIN dim_stores_gold dsg ON fsg.store_id = dsg.store_id
GROUP BY dsg.region, dp.category
ORDER BY dsg.region, total_sales_revenue DESC

Query Description

The analysis involves grouping the data by region and category, then sorting the categories within each region by total sales revenue in descending order to find the top-performing category.
