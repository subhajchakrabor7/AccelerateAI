Business Question: I want to use this data to identify the top-performing product categories by total sales revenue for each store region.

✅ Answer

The top-performing product categories by total sales revenue for each store region are: Midwest - Electronics ($9023.88), Northeast - Electronics ($17165.96), Southeast - Electronics ($9650.17), and West - Electronics ($28307.85).

📊 Approach & Query

Query Code

SELECT ds.region, dp.category, SUM(fs.total_amount) as total_sales_revenue
FROM fact_sales fs
JOIN dim_products dp ON fs.product_id = dp.product_id
JOIN dim_stores ds ON fs.store_id = ds.store_id
GROUP BY ds.region, dp.category
ORDER BY ds.region, total_sales_revenue DESC

Query Description

The analytical methodology used to reach this conclusion is to group the data by region and category, and then calculate the total sales revenue for each category in each region. The category with the highest total sales revenue in each region is then identified as the top-performing category.
